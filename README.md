# TechStore

##Description
This project is a POS (Point of Sale) & Inventory Management System developed by our team.

The system helps businesses manage products, track inventory, and handle sales transactions efficiently.

## Features
- Product management
- Inventory tracking
- Order & sales management
- Admin dashboard
- User authentication

## Technologies Used
- Backend: Node.js with MVC 
- Frontend: React.js / HTML, CSS, JavaScript 
- Database: MongoDB 

## Project Structure
- backend/
- frontend/
- admin/
